# Contributing to Free AI APIs

Thank you for helping keep this list accurate and useful!

## How to Add an Entry

1. **Fork** this repository
2. **Check** the entry doesn't already exist
3. **Add** your entry in alphabetical order within its category
4. **Verify** the free tier limits are accurate (link to the pricing page)
5. **Open a PR** with the title: `Add [Provider Name] to [Category]`

## Entry Format

```markdown
| **Provider Name** | Model name | Free limit description | ✅ or ❌ | Short note |
```

- Keep notes under 8 words
- Link provider name to their API signup page when possible
- Use ✅ for "no credit card required", ❌ for "credit card required"

## Updating Stale Entries

If a free tier has changed, open a PR with:
- Title: `Update [Provider] free tier limits`
- Link to the current pricing page as evidence

## Removing Discontinued Providers

If a free tier has been removed entirely:
- Title: `Remove [Provider] — free tier discontinued`
- Brief note in the PR description explaining what changed

## What We Don't Include

- Invite-only or waitlisted services
- Services with no documented rate limits
- Services that require a paid plan to do anything useful
- Promotional credits that last fewer than 14 days
